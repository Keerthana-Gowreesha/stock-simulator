# Stock Market Simulator

Fetches live BSE stock prices from Yahoo Finance using C, with a JavaScript frontend for buying/selling and portfolio tracking.

## Tech Stack
- C (data fetching via cURL)
- JavaScript (frontend logic)
- HTML/CSS (UI)
- Yahoo Finance API

## Features
- Live price fetch for 5 BSE stocks (RELIANCE, TCS, INFY, HDFCBANK, ICICIBANK)
- Buy/sell stocks with quantity selection
- Real-time profit/loss tracking per holding
- Refresh prices on demand
- Simulate next day with random price fluctuation (-8% to +8%)

### Prerequisites
